# Python 基础知识

## 变量与数据类型

Python 中的变量是对象的引用。Python 支持多种数据类型，包括整数（int）、浮点数（float）、字符串（str）、列表（list）、字典（dict）等。

## 函数定义

函数是可重用的代码块。使用 def 关键字定义函数。函数可以接收参数并返回值。

## 列表操作

列表是有序的可变集合。可以使用索引访问列表元素，使用 append() 方法添加元素，使用 pop() 方法删除元素。

## 字典操作

字典是键值对的无序集合。可以使用键访问值，使用 update() 方法更新字典，使用 keys() 和 values() 方法获取键和值。

## 条件语句

if 语句用于条件判断。可以使用 if、elif 和 else 关键字构建条件分支。

## 循环语句

for 循环用于遍历序列，while 循环用于条件循环。可以使用 break 和 continue 控制循环流程。
